## Overview
This project demonstrates data processing using Apache Spark and storage in MongoDB.

## Steps

### 1. Data Preparation
- Cleaned datasets
- Removed nulls and duplicates

### 2. Data Storage
- Converted DataFrames to JSON
- Stored in MongoDB collections:
  - customers
  - people
  - organizations

### 3. MongoDB Queries
- Customers per country
- Top 5 countries
- Customers per company
- Subscription date analysis

### 4. Visualization
- Bar charts (country, company)
- Pie chart (gender)
- Histogram (age, spending)
- Line chart (subscription trends)

## Technologies Used
- Apache Spark
- MongoDB Atlas
- PySpark
